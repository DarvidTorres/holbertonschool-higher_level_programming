| Operator | Example      | Same As         |
| -------- | ------------ | --------------- |
| `=`      | `x = 5`      | `x = 5`         |
| `+=`     | `x += 3`     | `x = x + 3`     |
| `-=`     | `x -= 3`     | `x = x - 3`     |
| `*=`     | `x *= 3`     | `x = x * 3`     |
| `/=`     | `x /= 3`     | `x = x / 3`     |
| `%=`     | `x %= 3`     | `x = x % 3`     |
| `//=`    | `x //= 3`    | `x = x // 3`    |
| `**=`    | `x **= 3`    | `x = x ** 3`    |
| `&=`     | `x &= 3`     | `x = x & 3`     |
| `^=`     | `x ^= 3`     | `x = x ^ 3`     |
| `>>=`    | `x >>= 3`    | `x = x >> 3`    |
| `<<=`    | `x <<= 3`    | `x = x << 3`    |


| <code>\|=</code>     | <code>x \|= 3</code> | <code>x = x \| 3</code> |
